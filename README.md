# Web Project

這是一個簡單的網頁專案，包含 HTML、CSS 和 JavaScript 檔案。

## 專案結構

```
web-project
├── index.html       # 專案的主 HTML 檔案
├── style.css        # 專案的樣式設定
└── script.js        # 專案的 JavaScript 程式碼
```

## 安裝步驟

1. 下載或克隆此專案。
2. 使用瀏覽器打開 `index.html` 檔案。

## 使用說明

- `index.html` 負責呈現網頁的結構和內容。
- `style.css` 定義了網頁的外觀樣式。
- `script.js` 處理網頁的互動和動態功能。
